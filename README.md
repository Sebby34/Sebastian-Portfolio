# Sebastian Portfolio Website

## Overview
This is my personal portfolio website, built to showcase my projects, skills, and contact information. The site is fully responsive and designed to provide a clean, professional presentation of my work.

## Live Demo
[View Live Portfolio](https://sebby34.github.io/Sebastian-Portfolio/)

## Features
- Responsive layout using **Flexbox** and **CSS Grid**
- Projects section highlighting personal projects with GitHub links
- Skills section with icons and descriptions for technologies I use
- Contact form for reaching out
- Clean and modern design for professional presentation

## Technologies Used
- **HTML5** – Semantic markup
- **CSS3** – Layout, Flexbox, Grid, styling
- **JavaScript** – Interactive elements and enhancements
- **Git/GitHub** – Version control
- **GitHub Pages** – Hosting
